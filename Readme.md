
This is an auto clicker for the DIVE web game at
https://alexfink.github.io/dive/

Uses pyautogui to send keypresses to the browser.

----------------------------------------------------------------------------

Press enter to exit the program.

Type a combination of WASD  characters to choose a custom (Up, Left, Down, Right) sequence to send.
Or choose an existing option from the menu.

This is a dumb, repetitive clicker.
